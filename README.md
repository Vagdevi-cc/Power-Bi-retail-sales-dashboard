#  Retail Sales Analytics Dashboard

## Overview

This project showcases an interactive Retail Sales Analytics Dashboard developed using Power BI. The dashboard provides insights into sales performance, profitability, customer segments, product categories, and regional sales trends through interactive visualizations and filters.

## Features

*  Total Sales Analysis
*  Profit Performance Tracking
*  Quantity Analysis
*  Discount Analysis
*  Regional Sales Insights
*  Customer Segment Analysis
*  Product Category Performance
*  Interactive Date Filtering
*  Dynamic Slicers and Visual Filters
*  Geographic Sales Visualization

## Tools Used

* Power BI
* Power Query
* Microsoft Excel
* Data Cleaning & Transformation
* Data Visualization

## Dashboard Highlights

* Interactive KPI Cards
* Sales Trend Analysis
* Category-wise Sales Distribution
* Customer Segment Insights
* Regional Performance Analysis
* Dynamic Filtering Capabilities

## Project Objective

The objective of this project is to transform retail sales data into meaningful business insights that support data-driven decision-making and performance monitoring.
